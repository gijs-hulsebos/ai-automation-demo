# 🚀 Enterprise AI Automation Engine & Portfolio

Welcome to my professional showcase. This platform is an advanced demonstration of **AI Orchestration**, **Autonomous Workflows**, and **Scalable AI Architecture**. 

The goal of this project is to bridge the gap between complex AI logic and intuitive, business-ready interfaces.

## ⚡ The "72-Hour" Challenge
This entire platform—from architecture and multi-language state management to the interactive workflow engine—was designed, developed, and deployed in **under 72 hours**. It serves as a case study in **AI-leveraged development efficiency**.

## 🛠️ Technical Stack
* **Frontend:** Next.js 15 (App Router), Tailwind CSS, Framer Motion.
* **AI Orchestration:** n8n (Advanced Event-Driven Workflows).
* **LLMs:** Google Gemini 1.5 Pro & OpenAI GPT-4o.
* **Backend & Data:** Supabase (PostgreSQL) & Webhooks.
* **Infrastructure:** Vercel Edge Network.

## 🌟 Key Features
* **Deep-Level Localization:** Full support for **English (EN)**, **Dutch (NL)**, and **German (DE)**. Unlike simple translation, the localization state is baked into the core, affecting everything from UI text to the real-time system logs.
* **Interactive System Trace:** A visual mockup of an AI automation pipeline. Users can follow a request as it moves through Webhooks, LLM Processing, Context Retrieval, and Final Output.
* **Live n8n Integration:** The contact forms and demo triggers are connected to live n8n workflows, demonstrating "Proof of Life" in production.
* **Premium SaaS UI:** Clean, dark-mode aesthetic focused on high-end enterprise standards and seamless user experience (UX).

## 📁 Repository Overview
* `/app`: Localized routing and page logic.
* `/components`: High-performance UI components including the dynamic language switcher and the interactive demo engine.
* `/public`: Media assets and production-ready infrastructure diagrams.
